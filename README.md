# 算法练习
此仓库用于算法练习作业提交

## Commit规范
此仓库下的commit严格按照以下格式提交：
> [姓名]-[本地测试情况]-[洛谷得分(如有)]
> 
> 例：Sam-本地测试通过-洛谷得分100

## Pull Request规范
PR请提交到对应分支(姓名)，请勿提交到main分支！

## Issue规范
如果遇到问题直接提issue。标题简洁描述所遇到的问题，详情在issue内容中提交，最好附带上代码

## 参考答案
参考答案会同步发布到[此仓库wiki页面](https://github.com/XGO-NOI/algorithm_assignments/wiki)
